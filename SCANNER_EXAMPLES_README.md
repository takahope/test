# 條碼掃描器模組範例集

本文檔包含從 `scanner_6.html` 拆分出的各個功能模組的獨立範例代碼。每個範例專注於特定功能，便於學習和整合。

## 📋 範例列表

### 1️⃣ 範例 01：基礎條碼掃描 (`example_01_basic_barcode_scanner.html`)

**功能說明：**
- 使用 Quagga2 庫進行即時條碼掃描
- 支援多種條碼格式（Code 128, EAN-13, EAN-8, Code 39, UPC-A, UPC-E）
- 簡單的啟動/停止控制
- 顯示掃描結果和條碼格式

**核心技術：**
- Quagga2 初始化與配置
- 攝影機權限管理
- 條碼偵測事件處理
- 媒體流資源清理

**使用場景：**
適合需要基本條碼掃描功能的應用程式

---

### 2️⃣ 範例 02：手動輸入條碼 (`example_02_manual_barcode_input.html`)

**功能說明：**
- 手動輸入條碼（支援鍵盤和掃碼槍）
- 條碼驗證與商品查找
- 掃描歷史記錄管理
- 商品資訊顯示（名稱、價格、條碼）

**核心技術：**
- 輸入框事件處理（Enter 鍵支援）
- 數據驗證與錯誤提示
- 動態列表渲染
- 記錄管理（添加/刪除/清空）

**使用場景：**
適合零售 POS 系統、庫存管理系統

---

### 3️⃣ 範例 03：多攝像頭切換 (`example_03_camera_switcher.html`)

**功能說明：**
- 自動偵測並列出所有可用攝像頭
- 動態切換前置/後置鏡頭
- 顯示當前攝像頭資訊（解析度、幀率、朝向）
- 平滑的切換過渡效果

**核心技術：**
- `navigator.mediaDevices.enumerateDevices()` API
- 攝像頭裝置識別與標籤處理
- Quagga 實例的動態重啟
- 媒體流完整釋放機制

**使用場景：**
適合需要在多個攝像頭之間切換的應用（如移動設備）

---

### 4️⃣ 範例 04：項目選擇管理 (`example_04_item_selector.html`)

**功能說明：**
- 可用項目與已選項目的雙欄管理
- 複選框選擇與取消
- 全選/清除全部功能
- 即時統計（總數量、總金額、類別分佈）

**核心技術：**
- 複選框狀態管理
- 動態 DOM 更新
- 數據聚合與統計
- 視覺高亮顯示

**使用場景：**
適合購物車、訂單管理、清單選擇等場景

---

### 5️⃣ 範例 05：掃描日誌記錄 (`example_05_scan_logger.html`)

**功能說明：**
- 即時日誌顯示（類似終端機風格）
- 分類統計（成功、錯誤、警告、重複）
- 詳細記錄表格
- 日誌匯出為 CSV

**核心技術：**
- 日誌分級管理
- 時間戳記錄
- 統計數據計算
- CSV 檔案生成與下載

**使用場景：**
適合需要審計追蹤、數據分析的系統

---

### 6️⃣ 範例 06：防重複掃描機制 (`example_06_duplicate_prevention.html`)

**功能說明：**
- 時間冷卻機制（可配置）
- 三種防重複模式（時間冷卻、嚴格阻擋、僅警告）
- 重複次數限制
- 冷卻進度條顯示

**核心技術：**
- `setTimeout` / `setInterval` 時間管理
- 掃描狀態追蹤
- 條碼去重邏輯
- 視覺化進度反饋

**使用場景：**
適合需要防止誤觸重複掃描的場景（如出貨檢查、簽到系統）

---

### 7️⃣ 範例 07：視覺反饋效果 (`example_07_visual_feedback.html`)

**功能說明：**
- 多種動畫效果（成功、錯誤、脈衝、彈跳、滑入、淡入）
- 浮動通知（Toast）
- 進度條效果
- 通知區域管理

**核心技術：**
- CSS 動畫（@keyframes）
- 動畫類別動態添加/移除
- 平滑滾動效果
- 非阻塞式通知

**使用場景：**
適合所有需要良好用戶體驗的應用

## 🛠️ 技術棧

- **前端框架：** 無（純 Vanilla JavaScript）
- **CSS 框架：** Tailwind CSS（通過 CDN）
- **條碼掃描：** Quagga2
- **瀏覽器 API：** MediaDevices, Canvas

## 📦 如何使用

### 基本使用

1. 將所需的 HTML 檔案複製到您的專案中
2. 在瀏覽器中打開檔案（需要支援 HTTPS 或 localhost）
3. 授予攝像頭權限（如果需要掃描功能）

### 整合到現有專案

每個範例都是獨立的，您可以：

1. **複製核心函數：** 將需要的 JavaScript 函數複製到您的專案
2. **提取 CSS 樣式：** 複製相關的 CSS 動畫和樣式
3. **修改 DOM 結構：** 根據您的需求調整 HTML 結構

### 組合使用

這些模組可以組合使用，例如：

```
範例 01 (基礎掃描) + 範例 06 (防重複) + 範例 07 (視覺反饋)
= 完整的掃描系統
```

---

## ⚠️ 注意事項

### 攝像頭權限

- 必須在 HTTPS 環境下運行（或 localhost）
- 首次使用需要用戶授予攝像頭權限

### 瀏覽器兼容性

- 建議使用現代瀏覽器（Chrome, Firefox, Safari, Edge）
- 部分功能需要支援 ES6+

### 移動設備

- 在移動設備上測試時，確保使用後置鏡頭
- 某些設備可能不支援多攝像頭切換

## 📝 自定義建議

### 樣式自定義
- 所有範例使用 Tailwind CSS，易於調整顏色和間距
- 可以替換為您自己的 CSS 框架

### 功能擴展
- 添加後端 API 整合
- 實現本地儲存（LocalStorage / IndexedDB）
- 添加離線支援（Service Worker）
- 整合其他條碼庫（如 ZXing）

### 性能優化
- 使用 Web Worker 處理條碼識別
- 實現虛擬滾動（長列表）
- 添加防抖/節流機制
